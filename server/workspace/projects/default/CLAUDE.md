# Default Project

This is your default UniSpace project. Files you create live in `files/`,
conversation history lives in `sessions/`, and project-local skills live in
`.claude/skills/`.

Edit this file to give your agent project-specific context, guidelines, or
personality. The agent reads `CLAUDE.md` at the start of every conversation.

## Guidelines

- Be concise and direct.
- Prefer minimal, precise changes.
- Explain reasoning only when the decision is non-obvious.
- Track multi-step work with the TodoWrite tool.
